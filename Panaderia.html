2019, Página web para Panadería local "La Esperanza"

Página web realizada con php y Aplicación realizada con Java para la venta de productos en una panaderia local, la pagina y la aplicación funcionaban de la misma manera, agregabas los productos que deseabas despues de haber creado una cuenta, los productos se quedaban guardados en tu cuenta y le notificaba al administrador (en este caso el dueño de la panaderia) que habias hecho un pedido, al momento de recogerlo pagabas.
